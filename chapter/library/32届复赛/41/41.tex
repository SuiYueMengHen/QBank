\begin{problem}[第32届全国中学生物理竞赛复赛][刚体碰撞与动能]{41}
    如图，在光滑水平桌面上有一长为 $L$ 的轻杆，轻杆两端各固定一质量均为 $M$ 的小球 A 和 B。开始时细杆静止；有一质量为 $m$ 的小球 C 以垂直于杆的速度 $v_{0}$ 运动，与 A 球碰撞。将小球和细杆视为一个系统。
    \begin{subproblem}
        求碰后系统的动能（用已知条件和球 C 碰后的速度表出）；
    \end{subproblem}
    \begin{subproblem}
        若碰后系统动能恰好达到极小值，求此时球 C 的速度和系统的动能。
    \end{subproblem}
\end{problem}
